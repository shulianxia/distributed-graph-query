# 分布式图查询系统 — test6（PyQt5 GUI 版）

## 概述

在 test3 预取版分布式架构上，用 **PyQt5** 实现了一个简洁的图形界面。

不同于 test1-test3 的命令行模式，test6 提供按钮 + 文本框 + 结果面板的可视化操作，零命令行输入负担。

## 功能

- **查看节点** — 输入节点 ID，显示标签、分组、度数、邻居列表
- **查询邻居** — 快速列出某节点的所有邻居节点
- **共同邻居** — 输入两个节点 ID，求交集
- **三角计数** — 单节点或全图三角计数（后台预取式 O(P) 算法）
- **全图统计** — 节点数、分区数、Worker 数、全图三角数
- **状态监控** — 每 3 秒自动轮询系统状态，底栏实时显示
- **暗色主题** — 深色护眼配色，字体清晰

## 依赖

- Python 3.6+
- PyQt5（`pip3 install PyQt5`）
- 零外部依赖（其余全部 Python 标准库）

## 启动方式

```bash
pip3 install PyQt5

# 一键启动（生成数据 → 启动服务 → 打开 GUI）
cd ~/桌面/ziliao/software/dazuoye/test6
python3 start_demo.py

# 自定义规模
python3 start_demo.py 500 0.06

# 仅启动服务，在另一终端手动打开 GUI
python3 start_demo.py --no-gui
python3 ui_app.py --coord-port 9000
```

## 与 test5 的对比

| 特性 | test5 (TUI) | test6 (GUI) |
|------|-------------|-------------|
| 交互方式 | 键盘输入命令 | 点击按钮 + 文本框 |
| 依赖 | 零（纯 stdlib） | PyQt5 |
| 并行查询 | 串行输入 | 后台线程，不阻塞界面 |
| 状态显示 | 文本滚动 | 实时自动刷新面板 |
| 视觉效果 | ANSI 彩色 | 暗色主题，等宽字体 |
