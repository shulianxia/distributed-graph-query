# test7 — 分布式图查询系统（多用户登录 + 美化 GUI）

基于 PyQt5 的分布式图查询系统 GUI，带多用户登录/切换、深色渐变主题、查询历史回溯。

## 功能特色

- 🛡️ **多用户系统** — 5 个预置用户（管理员/开发者/分析师/运维/访客），卡片式登录界面
- 🔗 **5种图查询** — 节点信息、邻居查询、共同邻居、三角计数、全图统计
- 🎨 **深色渐变主题** — 紫色系暗色主题，圆角卡片，hover 效果
- 📜 **查询历史** — 保存最近 20 条，双击回溯查看
- 👤 **用户切换** — 随时切换用户，历史记录保留
- ⚡ **异步查询** — 后台线程执行，不阻塞 UI

## 快速启动

```bash
cd test7
python3 start_demo.py
```

自动执行：
1. 清理旧进程
2. 生成测试数据（1000 节点，0.08 密度，5 分区，约 8 万条边）
3. 启动 Coordinator（:9000）
4. 启动 5 个 Worker（:9100-:9104）
5. 启动 GUI

也可手动启动：

```bash
# 终端1: 启动 Coordinator
python3 coordinator.py --port 9000

# 终端2: 启动 Workers
for i in {0..4}; do
    python3 worker.py --port $((9100+i))
done

# 终端3: 启动 GUI
python3 ui_app.py --coord-port 9000
```

## 文件说明

| 文件 | 说明 |
|------|------|
| `ui_app.py` | 入口文件，启动登录对话框 + 主界面 |
| `login_dialog.py` | 卡片式用户登录界面 |
| `main_window.py` | 主界面窗口（功能卡片 + 结果面板 + 历史） |

| `protocol.py` | 通信协议（复用 test6） |
| `coordinator.py` | 分布式 Coordinator（复用 test6） |
| `worker.py` | 查询 Worker（复用 test6） |
| `gen_all.py` | 测试数据生成器（复用 test6） |
| `start_demo.py` | 一键启动脚本 |
| `README.md` | 本文档 |
