# 分布式图查询系统 — test5（交互式 UI 版）

## 概述

在 test3 预取优化版的基础上，增加了**交互式终端 UI**，提供类似数据库 Shell 的友好操作体验。

与之前各版本的最大区别：不再需要每次敲 `python3 client.py neighbor 5` 这样的长命令，而是进入一个带 Tab 补全、彩色输出、命令历史的交互式环境。

## 交互式命令

| 命令 | 别名 | 说明 |
|------|------|------|
| `info <id>` | — | 节点详细信息（属性、标签、度数、邻居） |
| `neighbor <id>` | `n` | 查询邻居列表（紧凑显示） |
| `common <a> <b>` | `c` | 两个节点的共同邻居 |
| `triangle [id]` | `t` | 三角计数（无参数=全图预取，有参数=节点三角） |
| `stats` | `st` | 全图统计（分区数、节点数、三角数） |
| `edges [limit]` | — | 查看边列表 |
| `seek <keyword>` | — | 模糊搜索节点（按 ID 或标签） |
| `clear` | `cls` | 清屏 |
| `history` | `h` | 查看命令历史 |
| `help` | `?` | 显示帮助 |
| `shutdown` | — | 关闭 Coordinator |
| `exit` | `quit` | 退出客户端 |

## 架构

```text
               ┌─────────────────────┐
               │   TUI (ui.py)       │  ←── 交互式 Shell
               │   Tab 补全          │      彩色格式化输出
               │   命令历史          │      实时 RPC
               └────────┬────────────┘
                        │ TCP/JSON
               ┌────────▼────────────┐
               │  Coordinator        │  ←── 预取式三角计数
               │  :9000              │      多用户查询路由
               └──┬──┬──┬──┬──┬─────┘
                  │  │  │  │  │    TCP 短连接
               ┌──▼──▼──▼──▼──▼──┐
               │  Workers (5 个)  │  ←── 独立进程，各存分区
               │  :9100~9104      │      GraphStorage + LocalQuery
               └──────────────────┘
```

## 快速启动

```bash
cd test5/

# 一键体验（生成数据 → 启动服务 → 进入交互 UI）
python3 start_demo.py

# 自定义规模
python3 start_demo.py 500 0.06       # 500 节点，密度 6%
python3 start_demo.py 100 0.12       # 100 节点，高密度

# 只启动服务（在另一终端手动连接）
python3 start_demo.py --no-ui
# 然后在另一个终端：
python3 ui.py --coord-port 9000
```

## 交互式使用示例

启动后，你会在一个色彩丰富的 Shell 提示符 `dgraph >` 下操作：

```bash
dgraph > info 0
# → 显示节点 0 的详细信息（标签、度数、所有邻居）

dgraph > n 5
# → 显示节点 5 的邻居列表（紧凑排列）

dgraph > c 0 5
# → 显示节点 0 和 5 的共同邻居

dgraph > t
# → 预取全图三角计数（仅 5 次 RPC，毫秒级）

dgraph > t 3
# → 节点 3 参与的三角形

dgraph > stats
# → 全图统计概览

dgraph > seek server
# → 搜索标签包含 "server" 的所有节点

dgraph > h
# → 查看本次会话的命令历史
```

## 设计亮点

| 特性 | 实现 | 体验 |
|------|------|------|
| Tab 补全 | `readline` 模块 | 输入 `i` → Tab → 自动补全为 `info` |
| 彩色彩色 | ANSI 转义码 | 绿色=成功，红色=错误，青色=数据 |
| 表格对齐 | `print_table` | 节点/标签/度数对齐显示 |
| 命令历史 | `readline` 历史文件 | 跨会话持久化（`~/.dgraph_history`） |
| 预取三角 | 5次 RPC 拉全图 | 千节点规模毫秒级响应 |
| 多分区 | 一致性哈希 | 支持 5~8 分区 |

## 与 test3 的区别

| 特性 | test3（CLI 版） | test5（交互式 UI 版） |
|------|----------------|---------------------|
| 交互方式 | 每次 `python3 client.py <cmd>` | 实时 Shell：`dgraph >` |
| 输出格式 | 纯文本 JSON | 彩色格式化、表格、图标 |
| Tab 补全 | 无 | 有（`readline`） |
| 命令历史 | 无 | 有（跨会话持久化） |
| `info` 命令 | 无 | 有（节点详细信息） |
| `seek` 搜索 | 无 | 有（按标签/ID 模糊搜索） |
| `stats` 命令 | 无 | 有（全图统计概览） |
| `edges` 命令 | 无 | 有（边列表浏览） |
| `clear`/`history` | 无 | 有 |
| 确认机制 | 无 | shutdown 需确认 |
